¿Qué es Internet?

Red de computadoras que se encuentran interconectadas a nivel mundial para compartir información.


Tipos de Proveedores de Internet

ISP: Internet Service Provider, es el Proveedor de Servicios de Internet, es decir, a laempresa que vende a los usuarios la conexión a Internet.

IPP: Sistema basado en estándares para permitir la impresión remota desde un PC a cualquier impresora accesible.


¿Qué son los protocolos de Internet?

Conjunto de reglas que permiten que dos o más entidafes de un sistema de comuniquen entre ellos para transmitir información.


Interfaz de usuario (IX)

Medio por el cual una persona controla una aplicación de software o dispositivo de hardware (botones, enlaces, etc.).


Experiencia de usuario (UX)

La experiencia del usuario engloba todas aquellas interacciones del usuario con una marca (emociones).


¿Qué es www.?

World Wide Web (abreviada WWW) consiste en un complejo sistema de hipertextos e hipermedios interconectados entre sí y a los que puede tenerse acceso mediante una conexión a Internet y un conjunto de software especializado.


Nombres de dominio DNS

(Siglas Domain Name System) es un conjunto de protocolos y servicios que permite a los usuarios utilizar nombres en vez de tener que recordar direcciones IP numéricas.


¿Qué es hosting?

Servicio digital que tiene como función almacenar la información y contenido de un sitio web en centros de procesamiento de datos seguros y veloces.


¿Qué es y para qué sirve la semántica HTML?

Se refiere a la práctica de estructurar el código HTML de una página web de manera que el marcado (compuesto de las etiquetas y elementos) refleje el significado y la estructura del contenido de esta.

Sirve para que los buscadores y desarrolladores web puedan para leer y comprender con más facilidad, además suelen ser más accesibles para los lectores con deficiencias visuales y mejoran el posicionamiento en los buscadores.


¿Qué es SEO y su importancia?

Proviene de las siglas inglesas Search Engine Optimization, o lo que es lo mismo, la optimización de los motores de búsqueda. Así pues, una persona SEO viene a significar aquella cuyos conocimientos pueden posicionar un sitio web en los primeros resultados de los buscadores.


¿Qué es SEM y su importancia?

Search Engine Marketing, es el uso de herramientas y estrategias que ayudan a optimizar la visibilidad de las páginas web en los buscadores, a través de publicidad de pago.
